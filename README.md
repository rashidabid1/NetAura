NetAura is a **real-time digital devices tracking application** with a **premium dark-themed dashboard**. It monitors all devices on a network and works on **Android, iOS, and desktop devices**.

## Key Features
- Real-time device tracking with online/offline status  
- Device details: IP, MAC address, last seen  
- Search and filter devices  
- Network scan with live animation  
- Export device data (PDF/CSV)  
- Premium dark theme and responsive design  

---

## Tech Stack
- Frontend: HTML, CSS, JavaScript (or React)  
- Backend: Python (Flask/FastAPI) or Node.js  
- Database: SQLite or MySQL  
- Real-time updates: WebSockets or AJAX polling
